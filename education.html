Intro To Web Development

In this first semester class we learned about static web pages using the New Perspectives HTML/CSS textbook. Advanced Web Development

I took Advanced Web Design early because I enjoyed Intro to Web Design so much! In Advanced Web Design I used the same text book but mostly online resources such as W3Schools as instructed. I learned about DDT's, XML, bootstrap, mobile responsive design, XSD's and validation.

Java

In my second semester Java course we used the Head First Java textbook 2nd Edition. I learned much about Object Oriented Design including encapsulation, using abstract classes, inheritance, interfaces, overloaded constructors, and more. Advanced Java

This class was challenging but fun! We learned GIT and ANT commands while using Java 8 API documentation to build an application that takes an input stream, analyzes a .txt file, and then writes a new file detailing certain information about that file. We studied different collections in Java to help analyze the tokens of the file. Towards the end of the semester we learned how to use Java servlets and JSP pages. See the example in my portfolio section to view the user interface I built! We developed in a Linux image using TomCat.

PHP with MySQL

In this second semester course I learned about cookies, session data, self-referencing forms, switch statements, CAPTCHA authentication, RSS Feeds and API's, and MySQL script. This includes querying and inserting data into a database. We use the Head First Php with MySQL text book. Also, we have touched on security issues such as SQL injections, and utilizing security features such as web-authentication, SHA1, and password_hash.

Systems Design

In Systems Design we learned about how software projects are completed while experiencing team dynamics. I am excited to keep learning about agile methods and the software development life cycle!
