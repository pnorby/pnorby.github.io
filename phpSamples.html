PHP

This is an application to track exercise that I coded in PHP, HTML, CSS, and MySQL. I coded several scripts to re-use with almost every page in this application (not all are shown). Initiating the session and checking session data/cookies, the connection details, the HTML headers, and the navigation bar are all separate PHP scripts. This site uses bootstrap to help with the navigation bar and layout of content. To store the data, I created a database and two tables using MySQL. One of the tables holds user information and the other holds exercise entries which are referenced by their userID to display their specific entries on the View Profile page.
