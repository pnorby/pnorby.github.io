This project spanning 12 weeks was to develop an application that would process a .txt file and output different specs of the document. It allows you to search for key words, find the total number of distinct words, where a word is located in the document, and the counts of words by word length. The project emphasized collections, File I/O, .properties files, and exception handling. We used JSP pages and JSTL to display the information to the user.




This next project was the last major project of the course. It is an example of an employee database administrator app. We used a MySQL database to store/retrieve our data. We used Java servlets, GETs and POSTs, JSP pages, JSTL, Java Bean, and MVC architecture.
