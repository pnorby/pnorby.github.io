Java

The following are two samples of projects from my Advanced Java course. Please reachout to me if you'd like to discuss these further!


This project spanning 12 weeks was to develop an application that would process a .txt file and output different specs of the document. It allows you to search for key words, find the total number of distinct words, where a word is located in the document, and the counts of words by word length. The project emphasized collections, File I/O, .properties files, and exception handling. We used JSP pages and JSTL to display the information to the user.



This next project was the last major project of the course. It is an example of an employee database administrator app. We used a MySQL database to store/retrieve our data. We used Java servlets, GETs and POSTs, JSP pages, JSTL, Java Bean, and MVC architecture.

Introduction to Java course

This is a very small application I wrote to demonstrate some of the concepts I learned in Java such as inheritance and constructors. Most of the application code is in the "run" method in a different class which isn't shown (feel free to ask me any questions about it though!). This application is for people who would like to rent a pilot for a certain period of time. The application asks the user which type of pilot they'd like and for how many hours. Then, depending on which pilot they request, an object of that type is created using the hours that the user entered as a constructor. The application then calls a "displayGreeting" method which is declared in the abstract superclass "Pilot" on that object. Lastly, the application uses the overridden chargeCustomer() method to let the customer know what their bill will be. The application is run 3 times with different selections in the image above.
