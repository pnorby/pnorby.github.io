Where I've been

My first professional experience was as a .NET Developer at WPS Health Solutions. I used Visual Studio 2017 to work on a variety of batch applications, Windows Forms integrations, and an API. In this position I was also able to obtain my Dell Boomi Professional Developer certificate and work as a citizen developer on some Boomi integrations. In early 2021 I moved into a Java developer role creating web services, migrating data from DB2 to SQL Server, and generating Stored Procedures based on legacy Cobol programs.

I took a position as a contracted Full Stack Developer at Electronic Theater Controls in the fall of 2021. My responsibilities included maintaining the full stack of multiple C# web applications, a legacy Java application, and acting as lead developer for any new enterprise applications that the business requested.
